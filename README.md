# Generador de Videos Motivacionales

Una aplicación moderna y atractiva construida en Flutter que permite a los usuarios crear videos motivacionales a partir de sus imágenes. 

## Características
*   **Selección de Imágenes**: Interfaz intuitiva para subir o capturar la imagen base (actualmente emulada con un marcador visual).
*   **Simulación de IA**: Retroalimentación visual interactiva mientras el video "se genera".
*   **Visualizador de Resultados**: Reproductor de video de demostración para ver el resultado final de la motivación.
*   **Modo Oscuro Elegante**: Un diseño optimizado para resaltar el contenido visual con detalles en color ámbar.

## Tecnologías y Plataformas
Este proyecto está desarrollado puramente en Flutter, permitiendo su compilación e iteración sobre:
*   iOS y Android (Móviles)
*   Windows, macOS, Linux (Escritorio)
*   Web

## Cómo Ejecutarlo
Si cuentas con el SDK de Flutter instalado en tu sistema, sigue estos pasos:

1.  Abre la terminal en la raíz de este proyecto.
2.  Ejecuta `flutter pub get` para asegurar las dependencias.
3.  Ejecuta `flutter run` y elige tu dispositivo objetivo (Chrome, emulador, escritorio, etc.).
